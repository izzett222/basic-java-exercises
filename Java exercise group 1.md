### Main Focus
- Control Flow
- Operators
- Primitive Data Types
- Dealing with input and command-line arguments
### Supporting Topics
- Methods
- Strings
- Arrays
- Input handling

**For each Exercise, try not to put everything in the main method. Put them in a well organized class if you can.**


# Exercises
## Exercise 1
1. Ask the user for a word.
2. Ask the user for 3 numbers.
3. Calculate the sum of the numbers.
4. Calculate the average.
5. Display:
    - the largest number,
    - the smallest number.
6. Display whether the average is greater than 50.
7. If the average is greater than 80:
    - display `"Excellent"`.
8. If the average is between 50 and 80:
    - display `"Good"`.
9. Otherwise:
    - display `"Needs Improvement"`.
10. Display the word entered by the user in uppercase.
11. Display the length of the word.

## Exercise 2
1. Create a variable with an initial numeric value.
2. Ask the user for a number.
3. Reject the number if it is negative.
4. Ask them again until they give a non negative number
5. Subtract the number from the initial value.
6. Display the updated value.
7. Ask the user if they want to continue.
8. Repeat the process using a loop.
9. Count how many successful operations were performed.
10. Stop the program when the value becomes 0 or less.
11. Display the total number of operations.

## Exercise 3
1. Ask the user how many values they want to enter.
2. Use a loop to collect all values.
3. Calculate the total sum.
4. Calculate the average.
5. Count how many numbers are:
    - even,
    - odd,
    - greater than 100.
6. Display the largest value entered.
7. Display the smallest value entered.
8. Display all entered values in reverse order.
9. Display whether the average is above 50.

## Exercise 4
1. Ask the user for a word.
2. Check whether:
    - the word length is at least 8,
    - the word contains an uppercase letter,
    - the word contains a number.
3. Display which conditions failed.
4. Allow the user 3 attempts.
5. Stop immediately if all conditions pass.
6. Display `"Valid"` or `"Invalid"`.
7. Display the number of remaining attempts after each failure.

## Exercise 5
1. Ask the user for a number.
2. Use a loop to display all numbers from 1 up to that number.
3. Display:
    - the sum of all numbers,
    - the count of even numbers,
    - the count of odd numbers.
4. Display all numbers divisible by 3.
5. Display all numbers divisible by both 3 and 5.
6. Stop early if the loop reaches 100.
7. Display whether the entered number is positive, negative, or zero.

## Exercise 6
1. Read 2 numbers from command-line arguments.
2. Convert both values to integers.
3. Calculate sum, difference, product, and division.
4. Check if either number is zero.
5. If division is not possible, display `"Invalid division"`.
6. Determine which number is larger.
7. Display whether both numbers are positive.
8. Display whether at least one number is even.
9. Display the results in a formatted output string.

## Exercise 7
1. Create an array of integers: `{12, 5, 8, 21, 7, 3, 14}`.
2. Loop through the array.
3. Calculate total sum.
4. Calculate average.
5. Find max value.
6. Find min value.
7. Count numbers greater than 10.
8. Count even and odd numbers.
9. Replace all values below 5 with 0.
10. Display final modified array.

## Exercise 8
1. Ask user for a word.
2. Define a constant word in code: `"JAVA"`.
3. Compare both words (ignore case).
4. Display whether they are equal.
5. Combine both words into one string.
6. Display length of both words.
7. Display the first character of each word.
8. Check if either word contains letter `"a"`.
9. Convert both words to uppercase and display them.
10. Display the word with more characters.

## Exercise 9
1. Read all command-line arguments as integers.
2. Store them in an array.
3. Calculate sum of all values.
4. Calculate average.
5. Find largest number.
6. Find smallest number.
7. Count how many values are negative.
8. Count how many values are greater than average.
9. Reverse the array manually (no built-in methods).
10. Display final results.

## Exercise 10
1. Ask user for a number.
2. Define an array: `{2, 4, 6, 8, 10}`.
3. Multiply each array element by the user input number.
4. Store results in a new array.
5. Calculate total sum of new array.
6. Count how many values exceed 20.
7. Replace values greater than 50 with 50.
8. Display both original and modified arrays.
9. Check if user input is even or odd.
10. Display final computed summary.

## Exercise 11
1. Ask user for a number `n`.
2. Loop from 1 to `n`.
3. Skip numbers divisible by 3.
4. Print the number not skipped
5. Stop loop if number reaches 20 (even if n is larger).
6. Count printed numbers.
7. Count skipped numbers.
8. Sum all printed numbers.
9. Display whether loop ended naturally or stopped early.
10. Display average of printed numbers.
11. Display final counts.

## Exercise 12

2. Read a number from command-line arguments.
3. Repeat the word based on the number.
4. Display each repetition on a new line.
5. Count total characters printed.
6. Convert word to lowercase and uppercase.

## Exercise 13

4. If divisible by 2, subtract 1.
5. Otherwise, keep unchanged.
6. Store results in new array.
7. Count how many values changed.
8. Count how many stayed the same.
9. Calculate total sum.
10. Display final transformed array.

## Exercise 14
